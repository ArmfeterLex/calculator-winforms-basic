# Calculator WinForms Basic

Простой калькулятор на C# Windows Forms с базовыми операциями.

<img width="615" height="202" alt="image" src="https://github.com/user-attachments/assets/02390dce-4c7a-485d-b873-3505f4a2cbab" />

## Функциональность

- Поддерживаемые операции:
  - Сложение (+)
  - Вычитание (-)
  - Умножение (*)
  - Деление (/)
  - Возведение в степень (^)
- Два поля для ввода чисел
- Одно поле для результата
- Обработка ошибок ввода (некорректные символы)
- Защита от деления на ноль

## Интерфейс

- textBox1 — первое число
- textBox2 — второе число
- textBox3 — результат
- button1 — сложение
- button2 — вычитание
- button3 — умножение
- button4 — деление
- button5 — возведение в степень

## Пример работы

1. Введите первое число (например, 10)
2. Введите второе число (например, 3)
3. Нажмите нужную операцию:
   - При нажатии "+" результат: 13
   - При нажатии "-" результат: 7
   - При нажатии "*" результат: 30
   - При нажатии "/" результат: 3.333...
   - При нажатии "^" результат: 1000 (10³)

## Куда можно встроить

- В учебные проекты по C#
- В демонстрацию работы с формами и обработкой событий
- Как базовый компонент для более сложных калькуляторов
- В примеры обработки исключений

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Операция "^" реализована через Math.Pow. При делении на ноль выводится сообщение об ошибке, результат не отображается.
